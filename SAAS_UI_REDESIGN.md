# Future SaaS UI Redesign Needed

The current UI is acceptable for personal-use MVP operations but does not meet
paid SaaS product standards. A future redesign should restructure the app
around a sidebar layout, true tabbed campaign detail views, reusable UI
components, clearer metrics, better onboarding, and a more polished visual
system.

## Redesign Scope

- New information architecture
- True dashboard layout
- Proper sidebar/nav
- Campaign detail tabs that actually reduce page length
- Dedicated pages for Sheet setup, Sequence, Sending, Logs, Suppression
- Cleaner component system
- Status badges and metrics cards
- Better empty/error states
- More polished onboarding
- Possibly shadcn/ui or another component library
- More intentional visual identity
